# anthro-env 快速上手（3 分钟）

这份文档只讲最短路径：安装、初始化、切换。

## 背景：为什么会有这个工具

说人话版：

- Claude Code 真的很好用，拿来当写代码的命令行助手非常顺手。
- 但官方渠道有时不太友好：token 贵、部分地区不支持、还有账号策略带来的不确定性。
- 现实里很多人会接入兼容 Anthropic 协议的第三方网关和模型，来保证可用性、也降低成本。
- 麻烦就来了：环境变量经常要改来改去，特别容易乱，还浪费时间。

`anthro-env` 就是为这个痛点做的：把“切配置”这件事变成一件简单、稳定、少出错的小事。

## 1) 安装

```bash
brew install anthro-env/tap/anthro-env
```

## 2) 初始化（首次一次）

```bash
anthro-env init
source ~/.zshrc
```

按提示输入：

- `Profile name`（示例：`default`）
- `ANTHROPIC_BASE_URL`
- `ANTHROPIC_MODEL`（可选，直接回车跳过；不填时走网关/服务端默认模型）
- `ANTHROPIC_AUTH_TOKEN`（会存进 Keychain）

## 3) 日常切换

```bash
anthro-env menu
```

- 输入编号切换配置
- 直接回车 = 退出（默认选 `0`）

## 4) 常用命令

```bash
anthro-env profile ls
anthro-env profile current
anthro-env profile add <name>
anthro-env profile use <name>
anthro-env doctor
```

## 5) 出问题先跑这个

```bash
anthro-env doctor
```

如果提示 hook 未安装，执行：

```bash
anthro-env init
source ~/.zshrc
```
