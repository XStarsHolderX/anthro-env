# 配置演进与维护建议

## 背景

`anthro-env` 的定位是一个长期可维护的配置管理工具。
随着你接入更多网关和模型，配置会不断增加，这份文档给出简单、稳定的维护方式。

## 建议做法

1. profile 名称保持清晰
- 按用途命名，例如：`official`、`proxy-cn`、`team-prod`

2. 把 token 与配置分离
- token 放 Keychain
- profile 文件只放非敏感配置

3. 定期清理不用的 profile
- 用 `anthro-env profile ls` 查看
- 用 `anthro-env profile rm <name>` 清理

4. 每次大改后做健康检查
- 运行 `anthro-env doctor`
- 确认 hook、profile、Keychain 三项正常
